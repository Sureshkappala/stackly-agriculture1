FAQ

Common Questions

How are Stackly's data centers powered?
Every regional node is integrated directly within local wind or solar farm grids. We use dynamic compute offloading to adjust computational intensities during power generation surpluses, making our calculations carbon-neutral.
What sensors are compatible with your API?
Our edge nodes accept any MQTT or HTTP payload structures. We supply pre-written SDK libraries for Python, C++, and Javascript nodes, allowing you to establish telemetry tunnels within seconds.
Is telemetry stored securely on your server arrays?
Yes. All sensor data is encrypted at rest using AES-256 standard and in transit via TLS 1.3 tunnels. Database replication is distributed across independent geographic nodes to guarantee protection from failure.
